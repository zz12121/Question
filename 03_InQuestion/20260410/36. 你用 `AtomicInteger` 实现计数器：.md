36. 你用 `AtomicInteger` 实现计数器：
```java
AtomicInteger counter = new AtomicInteger(0);
public void increment() {
    counter.incrementAndGet();
}
```
**问题：** 如果要求"每秒最多 1000 次"，如何改造？如果多线程竞争严重，有什么替代方案？

---
## 先说结论

"每秒最多1000次"需要**滑动窗口限流**：用多个时间槽记录每秒请求数，原子更新当前槽计数。竞争严重时，**LongAdder**（分段累加）替代`AtomicInteger`，或**令牌桶/漏桶算法**平滑限流，分布式场景用**Redis+Lua**原子操作。

## 🔥 详细展开：滑动窗口限流实现

### 方案1：简单滑动窗口（AtomicInteger数组）

```java
@Component
public class SlidingWindowRateLimiter {
    
    // 10个时间槽，每槽100ms，共1秒窗口
    private static final int SLOT_COUNT = 10;
    private static final int SLOT_DURATION_MS = 100;
    private static final int LIMIT_PER_SECOND = 1000;
    private static final int LIMIT_PER_SLOT = LIMIT_PER_SECOND / SLOT_COUNT;
    
    private final AtomicInteger[] slots = new AtomicInteger[SLOT_COUNT];
    private final AtomicLong currentSlotIndex = new AtomicLong(0);
    
    public SlidingWindowRateLimiter() {
        for (int i = 0; i < SLOT_COUNT; i++) {
            slots[i] = new AtomicInteger(0);
        }
        // 启动槽滑动线程
        startSlotSliding();
    }
    
    /**
     * 尝试获取许可
     */
    public boolean tryAcquire() {
        long slotIndex = currentSlotIndex.get();
        int index = (int) (slotIndex % SLOT_COUNT);
        
        // 原子增加当前槽计数
        int count = slots[index].incrementAndGet();
        
        if (count > LIMIT_PER_SLOT) {
            // 超过限制，回滚并拒绝
            slots[index].decrementAndGet();
            return false;
        }
        
        return true;
    }
    
    /**
     * 槽滑动（每100ms执行一次）
     */
    private void startSlotSliding() {
        ScheduledExecutorService scheduler = Executors.newSingleThreadScheduledExecutor();
        scheduler.scheduleAtFixedRate(() -> {
            long nextSlot = currentSlotIndex.incrementAndGet();
            int index = (int) (nextSlot % SLOT_COUNT);
            slots[index].set(0);  // 重置新槽
        }, SLOT_DURATION_MS, SLOT_DURATION_MS, TimeUnit.MILLISECONDS);
    }
    
    /**
     * 获取当前窗口总请求数
     */
    public int getCurrentWindowCount() {
        int sum = 0;
        for (AtomicInteger slot : slots) {
            sum += slot.get();
        }
        return sum;
    }
}
```

### 方案2：精确滑动窗口（时间戳版）

```java
@Component
public class PreciseSlidingWindow {
    
    private static final int WINDOW_SIZE_MS = 1000;  // 1秒窗口
    private static final int LIMIT = 1000;
    
    // 使用ConcurrentLinkedQueue记录请求时间戳
    private final ConcurrentLinkedQueue<Long> timestamps = new ConcurrentLinkedQueue<>();
    
    public synchronized boolean tryAcquire() {
        long now = System.currentTimeMillis();
        long windowStart = now - WINDOW_SIZE_MS;
        
        // 清理窗口外的旧记录
        while (!timestamps.isEmpty() && timestamps.peek() < windowStart) {
            timestamps.poll();
        }
        
        // 检查当前窗口内请求数
        if (timestamps.size() >= LIMIT) {
            return false;
        }
        
        timestamps.offer(now);
        return true;
    }
}
// 注意：此方案需要同步，高并发下性能差，仅适合低QPS场景
```

## 🔥 详细展开：高并发竞争优化

### 优化1：LongAdder替代AtomicInteger

```java
@Component
public class LongAdderRateLimiter {
    
    // LongAdder分段累加，高并发下性能更好
    private final LongAdder counter = new LongAdder();
    private final AtomicLong windowStart = new AtomicLong(System.currentTimeMillis());
    private static final int LIMIT = 1000;
    private static final int WINDOW_MS = 1000;
    
    public boolean tryAcquire() {
        long now = System.currentTimeMillis();
        long start = windowStart.get();
        
        // 窗口滑动
        if (now - start >= WINDOW_MS) {
            if (windowStart.compareAndSet(start, now)) {
                counter.reset();  // 新窗口，重置计数
            }
        }
        
        counter.increment();
        
        // 注意：这里可能有精度问题，仅作示例
        return counter.sum() <= LIMIT;
    }
}
```

**LongAdder vs AtomicInteger：**

| 场景 | AtomicInteger | LongAdder |
|------|--------------|-----------|
| 低并发 | ✅ 性能好 | 略差 |
| 高并发 | ❌ 大量CAS失败 | ✅ 分段累加，无竞争 |
| 读取频率 | 实时准确 | 最终一致（sum时合并） |
| 适用 | 计数、序号 | 统计、限流 |

### 优化2：令牌桶算法（平滑限流）

```java
@Component
public class TokenBucketRateLimiter {
    
    // 令牌桶：容量1000，每秒产生1000个令牌
    private final AtomicInteger tokens = new AtomicInteger(1000);
    private final AtomicLong lastRefillTime = new AtomicLong(System.currentTimeMillis());
    
    private static final int CAPACITY = 1000;      // 桶容量
    private static final int RATE_PER_SECOND = 1000; // 每秒产生令牌数
    
    public boolean tryAcquire() {
        refill();  // 先补充令牌
        
        while (true) {
            int current = tokens.get();
            if (current <= 0) {
                return false;  // 无令牌，限流
            }
            // CAS扣减令牌
            if (tokens.compareAndSet(current, current - 1)) {
                return true;
            }
        }
    }
    
    /**
     * 补充令牌
     */
    private void refill() {
        long now = System.currentTimeMillis();
        long last = lastRefillTime.get();
        long elapsed = now - last;
        
        if (elapsed > 0) {
            int tokensToAdd = (int) (elapsed * RATE_PER_SECOND / 1000);
            if (tokensToAdd > 0) {
                if (lastRefillTime.compareAndSet(last, now)) {
                    // 原子更新令牌数，不超过容量
                    tokens.updateAndGet(current -> 
                        Math.min(CAPACITY, current + tokensToAdd));
                }
            }
        }
    }
}
```

@Service
public class DistributedRateLimiter {
    
    @Autowired
    private StringRedisTemplate redisTemplate;
    
    private static final String KEY_PREFIX = "rate_limit:";
    
    /**
     * Redis+Lua原子限流（滑动窗口）
     */
    public boolean tryAcquire(String key, int limit, int windowSeconds) {
        String redisKey = KEY_PREFIX + key;
        long now = System.currentTimeMillis();
        long windowStart = now - windowSeconds * 1000;
        
        // Lua脚本保证原子性
        String lua = 
            "redis.call('ZREMRANGEBYSCORE', KEYS[1], 0, ARGV[1]);" +  // 删除窗口外
            "local count = redis.call('ZCARD', KEYS[1]);" +           // 当前计数
            "if count < tonumber(ARGV[2]) then" +                     // 未超限
            "  redis.call('ZADD', KEYS[1], ARGV[3], ARGV[3]);" +      // 添加当前
            "  redis.call('EXPIRE', KEYS[1], ARGV[4]);" +             // 设置过期
            "  return 1;" +                                           // 允许
            "else" +
            "  return 0;" +                                           // 拒绝
            "end;";
        
        Long result = redisTemplate.execute(
            new DefaultRedisScript<>(lua, Long.class),
            Collections.singletonList(redisKey),
            String.valueOf(windowStart),    // ARGV[1]
            String.valueOf(limit),           // ARGV[2]
            String.valueOf(now),             // ARGV[3]
            String.valueOf(windowSeconds)    // ARGV[4]
        );
        
        return result != null && result == 1;
    }
}
