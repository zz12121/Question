## 题干

你这个学期必须选修 `numCourses` 门课程，标记为 `0` 到 `numCourses-1`。

在选修某些课程之前可能有一些先修要求，例如，想要学习课程 `0`，你需要先完成课程 `1`，我们用一个匹配来表示：`prerequisites[i] = [ai, bi]`，表示如果想要学习课程 `ai`，则必须先完成课程 `bi`。

判断是否可能完成所有课程的学习？如果可以，返回 `true`；否则，返回 `false`。

### 示例

```
输入：numCourses = 2, prerequisites = [[1,0]]
输出：true
解释：总共2门课程，先修课程[1,0]：想要学1，先完成0

输入：numCourses = 2, prerequisites = [[1,0],[0,1]]
输出：false
解释：存在循环依赖，无法完成
```
