## 题干

实现 `pow(x, n)`，即计算 `x` 的整数 `n` 次幂函数。

你需要对 `n` 是负数的情况进行处理。**不能使用库函数。**

### 示例

```
输入：x = 2.00000, n = 10
输出：1024.00000

输入：x = 2.10000, n = 3
输出：9.26100

输入：x = 2.00000, n = -2
输出：0.25000
解释：2^-2 = 1/2^2 = 0.25
```
