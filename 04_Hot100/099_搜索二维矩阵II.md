## 题干

编写一个高效的算法，在 `m x n` 矩阵 `matrix` 中搜索目标值 `target`。该矩阵具有以下特性：
- 每行中的整数从左到右升序排列
- 每列中的整数从上到下升序排列

### 示例

```
输入：matrix = [
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
], target = 5
输出：true
```
