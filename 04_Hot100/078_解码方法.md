## 题干

一条包含字母 `A-Z` 的消息通过以下映射进行**编码**：

```
'A' -> 1
'B' -> 2
...
'Z' -> 26
```

给定一个只含数字的字符串 `s`，计算共有多少种**解码方法**。

注意：`0` 不能单独解码。

### 示例

```
输入：s = "12"
输出：2
解释：可以是 "AB"(1,2) 或 "L"(12)

输入：s = "226"
输出：3
解释：可以是 "BZ"(2,26)、"VF"(22,6) 或 "BBF"(2,2,6)
```
