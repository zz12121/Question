## 题干

在给定的 `m x n` 网格 `grid` 中，每个单元格可以有三个值之一：
- `0` = 空单元格
- `1` = 新鲜橘子
- `2` = 腐烂的橘子

每分钟，腐烂的橘子会将新鲜橘子腐烂（上下左右四个方向）。返回直到单元格中不可能有新鲜橘子为止，所需的**最少分钟数**。

### 示例

```
输入：grid = [[2,1,1],[1,1,0],[0,1,1]]
输出：4

输入：grid = [[2,1,1],[0,1,1],[1,0,1]]
输出：-1
解释：右下角的橘子永远不会腐烂
```
