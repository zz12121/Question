# 最短路径模型

## 题目信息

| 属性 | 值 |
|------|-----|
| 关联题型 | #433 最小基因变化 / #127 单词接龙 |
| 难度 | 🟡 中等 |
| 类型 | BFS / 图 / 最短路径 |

## 题干

从起点到终点，每次只能改变一个字符，求**最少变换次数**。

常见变体：
- 基因变化（#433）：基因库序列变化
- 单词接龙（#127）：单词替换

核心思路：**BFS + 哈希**，将每一种状态看作图的一个节点。

### 示例（单词接龙）

```
输入：beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"]
输出：5
解释：hit -> hot -> dot -> dog -> cog
```
