## 题干

给你两个单词 `word1` 和 `word2`，请你计算将 `word1` 转换成 `word2` 所使用的最少操作数。

你可以在一个单词上进行以下三种操作：
- 插入一个字符
- 删除一个字符
- 替换一个字符

### 示例

```
输入：word1 = "horse", word2 = "ros"
输出：3
解释：
horse -> rorse (将 'h' 替换为 'r')
rorse -> rose (删除 'r')
rose -> ros (删除 'e')
```
