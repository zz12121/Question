## 题干

在二维平面上有 `n` 个气球，每个气球的水平直径在 `x` 轴上有一端点 `(x_start, 0)` 和另一端点 `(x_end, 0)`。一支箭可以选定一个 `x` 轴上的点 `x`，如果气球的水平直径满足 `x_start <= x <= x_end`，则气球会被引爆。

给定 `points` 数组（每个气球看作 `[x_start, x_end]`），返回引爆所有气球所需的最少箭数。

### 示例

```
输入：points = [[10,16],[2,8],[1,6],[7,12]]
输出：2
解释：两支箭可以引爆所有气球
```
