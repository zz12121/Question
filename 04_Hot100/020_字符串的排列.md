## 题干

给定两个字符串 `s1` 和 `s2`，写一个函数判断 `s2` 是否包含 `s1` 的排列。换句话说，判断 `s2` 中是否存在一个子串，是 `s1` 的全排列。

### 示例

```
输入：s1 = "ab", s2 = "eidbaooo"
输出：true
解释：s2 包含 "ba" 是 "ab" 的全排列

输入：s1 = "ab", s2 = "eidboaoo"
输出：false
```
