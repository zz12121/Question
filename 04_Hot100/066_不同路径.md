## 题干

一个机器人位于 `m x n` 网格的左上角，机器人每次只能**向下或向右**移动。机器人试图到达网格的右下角，共有多少条不同的路径？

### 示例

```
输入：m = 3, n = 7
输出：28

输入：m = 3, n = 2
输出：3
解释：从左上角到右下角有 3 条路径
```
