## 题干

给定一个二维整数数组 `envelopes`，其中每个元素 `[w, h]` 表示一个信封的宽度和高度。

如果一个信封可以装入另一个信封，则称这两个信封为"俄罗斯套娃"关系。请你找出**最多能有多少个**信封能组成一组俄罗斯套娃。

### 示例

```
输入：envelopes = [[5,4],[6,4],[6,7],[2,3]]
输出：3
解释：最多嵌套的信封数为 3，组合为 [2,3] => [5,4] => [6,7]
```
