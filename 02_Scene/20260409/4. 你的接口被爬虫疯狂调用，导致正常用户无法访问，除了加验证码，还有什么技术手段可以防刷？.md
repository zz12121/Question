## 先说结论

防爬虫不是单一手段，而是 **"识别 → 拦截 → 惩罚"** 的组合拳：

| 层次 | 手段 | 作用 |
|-----|------|------|
| **识别层** | UA/行为分析/IP画像/设备指纹 | 发现爬虫 |
| **拦截层** | 限流/熔断/验证码/黑名单 | 阻止访问 |
| **惩罚层** | 限速/封禁/返回假数据/计费 | 降低收益 |

**一句话答案**：通过多维度识别爬虫特征（行为、IP、设备） + 分层拦截限流 + 验证码挑战 + 假数据反爬，组合构建防爬体系。

### 1.1 基础特征识别

```java
@Component
public class BotDetector {
    
    @Autowired private RiskScoreService riskScoreService;
    
    public BotCheckResult check(HttpServletRequest request) {
        BotCheckResult result = new BotCheckResult();
        
        // 1. User-Agent 检测
        String userAgent = request.getHeader("User-Agent");
        if (isSuspiciousUA(userAgent)) {
            result.addRisk("SUSPICIOUS_UA", 30);
        }
        
        // 2. 常见爬虫特征
        if (isKnownBot(userAgent)) {
            result.addRisk("KNOWN_BOT", 100);  // 直接拦截
        }
        
        // 3. 无 UA 或异常 UA
        if (StringUtils.isBlank(userAgent) || isFakeUA(userAgent)) {
            result.addRisk("NO_UA", 50);
        }
        
        // 4. Accept-Language 缺失
        if (request.getHeader("Accept-Language") == null) {
            result.addRisk("NO_ACCEPT_LANGUAGE", 20);
        }
        
        // 5. Referer 缺失（正常浏览器访问一般会带）
        if (request.getHeader("Referer") == null) {
            result.addRisk("NO_REFERER", 25);
        }
        
        return result;
    }
    
    // 已知爬虫 UA 直接拦截
    private boolean isKnownBot(String ua) {
        if (ua == null) return false;
        String[] botPatterns = {
            "curl", "wget", "python-requests", "scrapy", "spider", 
            "bot", "crawl", "apache-httpclient", "okhttp", "httpclient"
        };
        String lowerUA = ua.toLowerCase();
        return Arrays.stream(botPatterns).anyMatch(lowerUA::contains);
    }
    
    private boolean isSuspiciousUA(String ua) {
        if (ua == null) return false;
        // UA 太长可能是伪造
        return ua.length() > 500;
    }
}
```

### 1.2 行为分析识别

```java
@Service
@Slf4j
public class BehaviorAnalyzer {
    
    @Autowired private RedisTemplate<String, Object> redis;
    
    private static final String UA_BEHAVIOR_KEY = "ua:behavior:";
    private static final String IP_BEHAVIOR_KEY = "ip:behavior:";
    
    /**
     * 记录用户行为
     */
    public void recordAccess(String fingerprint, String api, long responseTime) {
        String key = UA_BEHAVIOR_KEY + fingerprint;
        
        // 1. 请求次数 +1
        redis.opsForHash().increment(key, "request_count", 1);
        // 2. 响应时间
        redis.opsForHash().put(key, "last_response_time", responseTime);
        // 3. 最后访问时间
        redis.opsForHash().put(key, "last_access_time", System.currentTimeMillis());
        // 4. 记录访问的 API 集合
        redis.opsForSet().add(key + ":apis", api);
        
        // 设置过期
        redis.expire(key, 24, TimeUnit.HOURS);
    }
    
    /**
     * 分析行为特征，计算爬虫概率
     */
    public BehaviorProfile analyze(String fingerprint) {
        String key = UA_BEHAVIOR_KEY + fingerprint;
        
        Map<Object, Object> data = redis.opsForHash().entries(key);
        if (data.isEmpty()) {
            return BehaviorProfile.normal();
        }
        
        BehaviorProfile profile = new BehaviorProfile();
        
        // 1. 请求频率：正常用户 < 100/小时
        long requestCount = Long.parseLong(data.getOrDefault("request_count", "0").toString());
        profile.setRequestPerHour(requestCount);
        if (requestCount > 500) {
            profile.addSignal(100, "请求频率异常高");
        } else if (requestCount > 200) {
            profile.addSignal(60, "请求频率偏高");
        }
        
        // 2. 访问页面集中度：正常用户一般访问多个页面
        Long apiCount = redis.opsForSet().size(key + ":apis");
        profile.setApiCount(apiCount.intValue());
        if (apiCount < 5 && requestCount > 100) {
            profile.addSignal(40, "只访问少量API，可能是定向爬虫");
        }
        
        // 3. 访问时间间隔：爬虫通常间隔均匀或极短
        List<Long> intervals = getAccessIntervals(data);
        if (isUniformInterval(intervals)) {
            profile.addSignal(80, "访问间隔过于均匀，疑似机器访问");
        }
        
        // 4. 访问路径分析：正常用户有跳转逻辑
        Set<String> apis = redis.opsForSet().members(key + ":apis");
        if (isSuspiciousAccessPattern(apis)) {
            profile.addSignal(50, "访问路径不符合正常用户行为");
        }
        
        return profile;
    }
    
    // 检测访问间隔是否过于均匀（机器特征）
    private boolean isUniformInterval(List<Long> intervals) {
        if (intervals.size() < 5) return false;
        
        // 计算标准差
        double avg = intervals.stream().mapToLong(Long::longValue).average().orElse(0);
        double variance = intervals.stream()
            .mapToDouble(i -> Math.pow(i - avg, 2))
            .average().orElse(0);
        double stdDev = Math.sqrt(variance);
        
        // 标准差小于平均值 10%，说明间隔很均匀
        return stdDev < avg * 0.1;
    }
}
```

```java
@Service
public class IpRiskService {
    
    @Autowired private RedisTemplate<String, Object> redis;
    
    /**
     * IP 风险评分
     */
    public IpRiskProfile getIpRiskProfile(String ip) {
        IpRiskProfile profile = new IpRiskProfile();
        
        // 1. 查询 IP 归属地
        IpInfo ipInfo = ipGeoService.query(ip);
        profile.setCountry(ipInfo.getCountry());
        profile.setIsp(ipInfo.getIsp());
        
        // 2. 查询历史风险记录
        Long riskCount = redis.opsForZSet()
            .zCard("ip:risk:history:" + ip);
        profile.setHistoricalRiskCount(riskCount.intValue());
        
        // 3. 查询当前时间段请求量
        Long currentQps = redis.opsForValue().increment("ip:qps:" + ip);
        redis.expire("ip:qps:" + ip, 60, TimeUnit.SECONDS);
        
        // 4. 查询代理/IDC IP 库
        if (ipProxyChecker.isProxy(ip) || ipProxyChecker.isIDC(ip)) {
            profile.addRiskType(RiskType.PROXY_OR_IDC);
        }
        
        // 5. 计算综合评分
        int score = calculateRiskScore(profile);
        profile.setRiskScore(score);
        
        return profile;
    }
    
    private int calculateRiskScore(IpRiskProfile profile) {
        int score = 0;
        
        // 代理/IPC 直接 +80 分
        if (profile.getRiskTypes().contains(RiskType.PROXY_OR_IDC)) {
            score += 80;
        }
        
        // 历史风险记录
        score += Math.min(profile.getHistoricalRiskCount() * 10, 30);
        
        // 高风险地区
        if (profile.isHighRiskRegion()) {
            score += 20;
        }
        
        return Math.min(score, 100);
    }
}
```

```java
@Configuration
public class AntiCrawlerConfig {
    
    @Bean
    public FilterRegistrationBean<AntiCrawlerFilter> antiCrawlerFilter() {
        FilterRegistrationBean<AntiCrawlerFilter> bean = new FilterRegistrationBean<>();
        bean.setFilter(new AntiCrawlerFilter());
        bean.addUrlPatterns("/api/*");
        bean.setOrder(1);
        return bean;
    }
}

@Component
@Slf4j
public class AntiCrawlerFilter extends OncePerRequestFilter {
    
    @Autowired private BotDetector botDetector;
    @Autowired private BehaviorAnalyzer behaviorAnalyzer;
    @Autowired private IpRiskService ipRiskService;
    @Autowired private CaptchaService captchaService;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                   HttpServletResponse response,
                                   FilterChain chain) throws Exception {
        
        // 1. 获取客户端标识
        String ip = getClientIp(request);
        String fingerprint = request.getHeader("X-Device-Fingerprint");
        String userId = getUserId(request);  // 已登录用户
        
        // 2. 快速路径：已登录用户放行
        if (StringUtils.isNotBlank(userId)) {
            chain.doFilter(request, response);
            return;
        }
        
        // 3. IP 风险检查
        IpRiskProfile ipRisk = ipRiskService.getIpRiskProfile(ip);
        if (ipRisk.getRiskScore() > 80) {
            log.warn("[IP风险拦截] ip={}, score={}", ip, ipRisk.getRiskScore());
            writeResponse(response, 403, "访问被拒绝");
            return;
        }
        
        // 4. Bot 特征检查
        BotCheckResult botCheck = botDetector.check(request);
        if (botCheck.isDefinitelyBot()) {
            log.warn("[Bot拦截] ip={}, ua={}", ip, request.getHeader("User-Agent"));
            // 触发验证码
            captchaService.requireCaptcha(response, ip);
            return;
        }
        
        // 5. 行为分析（异步，不阻塞）
        CompletableFuture.runAsync(() -> {
            String uaKey = fingerprint != null ? fingerprint : ip;
            behaviorAnalyzer.recordAccess(uaKey, request.getRequestURI(), 
                System.currentTimeMillis());
            
            BehaviorProfile profile = behaviorAnalyzer.analyze(uaKey);
            if (profile.getRiskScore() > 70) {
                log.warn("[行为分析拦截] fingerprint={}, profile={}", 
                    uaKey, profile);
                captchaService.requireCaptcha(response, ip);
            }
        });
        
        // 6. 通用限流
        if (!rateLimiter.tryAcquire("global:" + ip, 1000, 1)) {
            response.setHeader("X-RateLimit-Limit", "1000");
            response.setHeader("X-RateLimit-Remaining", "0");
            response.setHeader("Retry-After", "60");
            writeResponse(response, 429, "请求过于频繁");
            return;
        }
        
        chain.doFilter(request, response);
    }
}
```

### 3.1 多级验证码策略

```java
@Service
public class CaptchaService {
    
    // 验证码难度分级
    enum CaptchaLevel {
        LIGHT,    // 简单：图片选择
        MEDIUM,   // 中等：滑块拼图
        HEAVY     // 复杂：点选文字/短信
    }
    
    /**
     * 根据风险等级返回不同验证码
     */
    public void requireCaptcha(HttpServletResponse response, String ip) 
            throws IOException {
        int riskLevel = calculateRiskLevel(ip);
        
        CaptchaLevel level = CaptchaLevel.values()[riskLevel];
        String captchaToken = generateCaptchaToken(level);
        
        // 保存 token 关联信息
        redisTemplate.opsForValue().set(
            "captcha:token:" + captchaToken,
            JSON.toJSONString(new CaptchaContext(ip, level)),
            5, TimeUnit.MINUTES
        );
        
        // 返回验证码页面
        response.setContentType("application/json");
        response.getWriter().write(JSON.toJSONString(new CaptchaChallenge(
            captchaToken,
            getCaptchaConfig(level)
        )));
    }
    
    /**
     * 验证验证码
     */
    public boolean verify(String captchaToken, String answer) {
        String contextJson = redisTemplate.opsForValue()
            .get("captcha:token:" + captchaToken);
        
        if (contextJson == null) {
            return false;
        }
        
        CaptchaContext context = JSON.parseObject(contextJson, CaptchaContext.class);
        
        // 验证逻辑
        boolean success = verifyWithProvider(context.getLevel(), answer);
        
        if (success) {
            // 验证成功，标记 IP/设备可信
            markTrusted(context.getIp());
            redisTemplate.delete("captcha:token:" + captchaToken);
        }
        
        return success;
    }
    
    // 标记 IP 可信（短期免验证）
    private void markTrusted(String ip) {
        String key = "ip:trusted:" + ip;
        redisTemplate.opsForValue().set(key, "1", 30, TimeUnit.MINUTES);
    }
}
```

```java
@Service
public class DataPoisoningService {
    
    /**
     * 对可疑爬虫返回假数据
     * 目的：污染爬虫数据，让攻击者无法分辨真假
     */
    public Object injectFakeData(Object realData, String clientIp) {
        // 1. 判断是否可疑
        if (!isSuspiciousClient(clientIp)) {
            return realData;
        }
        
        // 2. 按比例投毒（比如 30% 的数据是假的）
        if (Math.random() < 0.3) {
            return createFakeData(realData);
        }
        
        return realData;

        // 随机修改价格、数量等关键字段
        if (realData instanceof Product) {
            Product fake = ((Product) realData).clone();
            fake.setPrice(fake.getPrice().multiply(BigDecimal.valueOf(1.5)));
            fake.setStock(fake.getStock() + new Random().nextInt(100));
            return fake;
        }
        return realData;
    }
}
```

```java
@Service
public class FrequencyBanService {
    
    /**
     * 渐进式封禁
     */
    public BanResult processRequest(String ip, String api) {
        String key = "ban:request:" + ip + ":" + api;
        
        Long count = redisTemplate.opsForValue().increment(key);
        redisTemplate.expire(key, 60, TimeUnit.SECONDS);  // 1分钟窗口
        
        if (count == null) return BanResult.allow();
        
        // 分级惩罚
        if (count > 1000) {
            // 永久封禁
            addToBlacklist(ip, Duration.ofDays(365));
            return BanResult.permanentBan();
        } else if (count > 500) {
            // 封禁 1 小时
            redisTemplate.opsForSet().add("ban:1h", ip);
            return BanResult.temporaryBan(Duration.ofHours(1));
        } else if (count > 200) {
            // 限速 1 QPS
            return BanResult.rateLimit(1);
        }
        
        return BanResult.allow();
    }
    
    private boolean isSuspiciousClient(String ip) {
        // 综合判断逻辑
        Long riskScore = redisTemplate.opsForValue().get("ip:risk:" + ip);
        return riskScore != null && riskScore > 70;
    }
}
```

## 六、技术对比总结

| 手段 | 优点 | 缺点 | 适用场景 |
|-----|------|------|---------|
| **UA 检测** | 简单快速 | 可伪造 | 初筛 |
| **IP 限流** | 有效 | 误杀正常用户 | 通用 |
| **验证码** | 准确 | 影响体验 | 高风险请求 |
| **设备指纹** | 难以伪造 | 需要前端配合 | 精准识别 |
| **行为分析** | 智能 | 实现复杂 | 高级防护 |
| **假数据投毒** | 反向打击 | 可能误伤 | 对抗性爬虫 |
| **JS 加密挑战** | 拦截简单脚本 | 可被逆向 | 中等防护 |

## 面试加分话术

> "防爬虫的核心是 **让爬虫的成本高于收益**。
> 
> 我的实战经验：
> 1. **分层防御**：WAF 挡 50%，网关限流挡 30%，业务层行为分析挡 15%，验证码兜底 5%
> 2. **组合策略**：单一手段很容易被绕过，必须多维度配合
> 3. **动态调整**：爬虫会进化，防护策略也要动态更新
> 4. **业务结合**：比如价格接口返回假数据，污染爬虫数据质量
> 
> 之前我们被爬虫把商品数据薅光了，换了 3 种方案才搞定——最终是 **设备指纹 + 行为分析 + 渐进式验证码** 的组合拳，把爬虫请求量降了 95%。"

---

**一句话收尾**：防爬虫没有银弹，必须是 **"识别 + 拦截 + 惩罚 + 反爬"** 的多层次体系，让爬虫作者投入的成本远超他能获取数据的价值。
