# 理论综合题 - 第5天（121-150）

## Java 基础（5题）

121. 什么是 Java 的内部类？静态内部类、成员内部类、局部内部类、匿名内部类有什么区别？

122. Java 的 `Enum` 枚举本质是什么？枚举可以实现接口吗？枚举的单例模式有什么优势？

123. `Optional` 是什么？如何避免 NPE？`Optional` 的正确使用姿势是什么？

124. Java 中 `==` 和 `equals()` 的区别？Integer 缓存池（-128~127）是怎么回事？

125. 什么是 Java 的协变返回类型？什么是方法的重载和重写？有什么区别？

## Java 并发（6题）

126. 什么是 `LockSupport`？`park()` 和 `unpark()` 的原理是什么？和 `wait()/notify()` 有什么区别？

127. 什么是 `Condition`？和 `Object` 的 `wait()/notify()` 有什么区别？

128. 什么是 `ForkJoinPool`？工作窃取算法（Work-Stealing）是如何工作的？

129. `Executors.newCachedThreadPool()` 有什么问题？为什么阿里规约禁止使用 `Executors` 创建线程池？

130. 什么是线程的中断（Interrupt）？`interrupt()`、`isInterrupted()`、`interrupted()` 有什么区别？

131. 什么是 `volatile` 的 happens-before 规则？写操作对读操作的可见性如何保证？

## JVM（5题）

132. 什么是 JVM 的对象分配策略？什么情况下对象会直接分配到老年代？

133. 什么是 Minor GC、Major GC、Full GC？分别在什么情况下触发？

134. 什么是 JVM 的元空间（Metaspace）溢出？如何排查和解决？

135. 什么是 JVM 的字节码增强？ASM、Javassist、ByteBuddy 分别是什么？

136. 什么是 JVM 的 Attach 机制？Arthas 的工作原理是什么？
