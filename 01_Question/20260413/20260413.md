# 理论综合题 - 第5天（121-150）

## Java 基础（5题）

121. 什么是 Java 的内部类？静态内部类、成员内部类、局部内部类、匿名内部类有什么区别？匿名内部类为什么只能使用外部类的 final 或 effectively final 变量？

122. Java 的 `Enum` 枚举本质是什么？枚举可以实现接口吗？枚举的单例模式有什么优势？和双重检查锁单例相比有什么区别？

123. `Optional` 是什么？如何避免 NPE？`Optional` 的正确使用姿势是什么？为什么不能把 Optional 作为方法的参数或字段类型？

124. Java 中方法调用的底层机制是怎样的？静态绑定（早绑定）和动态绑定（晚绑定）有什么区别？虚方法表（vtable）是如何工作的？

125. 什么是 Java 的协变返回类型？什么是桥接方法（Bridge Method）？泛型擦除后为什么需要生成桥接方法？

## Java 并发（6题）

126. 什么是 `LockSupport`？`park()` 和 `unpark()` 的原理是什么？和 `wait()/notify()` 有什么区别？为什么 AQS 底层用 LockSupport 而不是 wait/notify？

127. 什么是 `Condition`？`ReentrantLock.newCondition()` 创建的 Condition 对象和 Object 的 `wait()/notify()` 有什么区别？如何实现一个有界阻塞队列？

128. 什么是 `ForkJoinPool`？工作窃取算法（Work-Stealing）是如何工作的？`commonPool()` 有什么坑？什么任务不适合 ForkJoinPool？

129. `Executors.newCachedThreadPool()` 有什么问题？为什么阿里规约禁止使用 `Executors` 创建线程池？如果必须用线程池，核心参数怎么定？

130. 什么是线程的中断（Interrupt）机制？`interrupt()`、`isInterrupted()`、`interrupted()` 三个方法分别做了什么？为什么调用 interrupt() 不一定能停止线程？

131. 什么是 Java 内存模型中的顺序一致性（Sequential Consistency）？为什么说即使加了 volatile 或 synchronized，程序也不一定是"严格按代码顺序执行"的？

## JVM（5题）

132. 什么是 JVM 的对象分配策略（对象优先在 Eden、大对象直接进老年代、长期存活对象进入老年代）？对象年龄是如何计算的？什么是动态年龄判定？

133. JVM 的垃圾收集器中，Parallel Scavenge 和 CMS 在设计目标上有什么本质区别？什么时候该选 CMS，什么时候该选 Parallel？

134. 什么是 JVM 的元空间（Metaspace）溢出？哪些场景会导致 Metaspace OOM？和 PermGen OOM 有什么不同？如何设置 Metaspace 大小？

135. 什么是 JVM 的字节码增强技术？ASM、Javassist、ByteBuddy 三种框架各有什么特点？Spring AOP 用的是哪种方式？

136. 什么是 JVM 的 Attach API？`jstack`、`jmap`、`jstat` 这些工具的工作原理是什么？Arthas 是怎么做到在线查看运行时信息的？

## MySQL（5题）

137. MySQL InnoDB 引擎中，`SELECT ... FOR UPDATE` 加的是行锁还是间隙锁？什么情况下会退化为表锁？RC 和 RR 隔离级别下加锁范围有什么不同？

138. 什么是 MySQL 的 Online DDL（在线DDL）？`ALTER TABLE ADD COLUMN` 时，MySQL 8.0 和 5.6 分别用什么算法实现？`ALGORITHM=INPLACE` 和 `ALGORITHM=COPY` 有什么区别？

139. MySQL 的 `VARCHAR(N)` 中的 N 到底代表什么？字符还是字节？`utf8mb4` 下 VARCHAR(255) 最大占用多少字节？行长度限制 65535 字节是怎么算出来的？

140. 什么是 MySQL 的自适应哈希索引（AHI）？它是在内存层面做的优化还是在存储引擎层面？什么情况下 AHI 反而会降低性能？如何关闭？

141. 什么是 Change Buffer（变更缓冲区）？它对哪些二级索引操作有效？为什么说写多读少的场景下 Change Buffer 能大幅提升性能？`innodb_change_buffer_max_size` 该怎么设？

## Redis（4题）

142. Redis 的内存淘汰策略中，`allkeys-lru` 和 `volatile-lfu` 怎么选？如果你的业务是"热点数据访问集中、冷数据几乎不被访问"和"所有数据访问频率差不多"两种情况，分别选哪种？

143. Redis 的大 key 问题：如果一个 Hash 有 100 万个 field，算不算大 key？大 key 会带来哪些性能隐患？如何用 `MEMORY USAGE` 和 `SCAN` 命令发现大 key？

144. Redis 的客户端缓冲区（client output buffer limit）有什么作用？如果一个客户端订阅了 Pub/Sub 但消费很慢，会发生什么？怎么防范"客户端缓慢导致 Redis 内存暴涨"?

145. Redis 的 `UNLINK` 命令和 `DEL` 命令有什么区别？为什么删除大 key 要用 UNLINK？它的后台异步释放是如何实现的？lazyfree 相关参数有哪些？
