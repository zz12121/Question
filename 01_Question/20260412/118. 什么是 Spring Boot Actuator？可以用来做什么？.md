## 先说结论

Spring Boot Actuator 是 Spring Boot 提供的**生产级监控和管理端点**，通过 HTTP/JMX 暴露应用内部运行状态，包括**健康检查、指标监控、线程状态、环境配置**等，是**微服务运维**的核心工具。

---

## 🔥 核心概念

| 维度 | 说明 |
|------|------|
| **本质** | 内置监控端点，暴露应用运行时信息 |
| **访问方式** | HTTP 端点（REST API）或 JMX |
| **核心功能** | 健康检查、指标收集、配置查看、线程 dump、日志级别调整 |
| **安全控制** | 通过 Spring Security 限制敏感端点访问 |
| **集成生态** | Micrometer（指标）、Prometheus（监控）、Grafana（可视化） |

### 1. 基础配置与启动

```xml
<!-- pom.xml -->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>
```

```yaml
# application.yml
management:
  endpoints:
    web:
      exposure:
        include: health,info,metrics,prometheus,env,beans,configprops,loggers,threaddump,heapdump
  endpoint:
    health:
      show-details: always  # 显示详细健康信息
    metrics:
      enabled: true
  server:
    port: 8081  # 管理端口分离（生产推荐）

| 端点 | 路径 | 功能 |
|------|------|------|
| **health** | `/actuator/health` | 应用健康状态 |

| **mappings** | `/actuator/mappings` | 所有 HTTP 端点映射 |

// 自定义健康检查
@Component
public class DatabaseHealthIndicator implements HealthIndicator {
    
    @Autowired
    private DataSource dataSource;
    
    @Override
    public Health health() {
        try (Connection conn = dataSource.getConnection()) {
            if (conn.isValid(1)) {
                return Health.up()
                    .withDetail("database", "MySQL")
                    .withDetail("connection", "valid")
                    .build();
            }
        } catch (SQLException e) {
            return Health.down()
                .withDetail("database", "MySQL")
                .withDetail("error", e.getMessage())
                .build();
        }
        return Health.down().build();
    }
}
```

## 🔥 扩展追问 2：指标监控（Micrometer）

```java
// 自定义指标
@Service
public class OrderService {
    
    private final Counter orderCounter;
    private final Timer orderTimer;
    private final Gauge stockGauge;
    
    public OrderService(MeterRegistry meterRegistry) {
        // 计数器
        this.orderCounter = Counter.builder("orders.created")
            .description("Total orders created")
            .register(meterRegistry);
        
        // 计时器
        this.orderTimer = Timer.builder("orders.processing.time")
            .description("Order processing time")
            .register(meterRegistry);
        
        // 仪表盘（动态值）
        this.stockGauge = Gauge.builder("inventory.stock", 
                this, OrderService::getCurrentStock)
            .description("Current stock level")
            .register(meterRegistry);
    }
    
    public void createOrder(Order order) {
        orderCounter.increment();
        
        orderTimer.record(() -> {
            // 处理订单逻辑
            processOrder(order);
        });
    }
}
```

# 恢复 INFO 级别

  -H "Content-Type: application/json" \
  -d '{"configuredLevel": "INFO"}'

**生产环境排查问题神器**：
```java
@RestController
@RequestMapping("/admin")
public class LogController {
    
    @Autowired
    private LoggersEndpoint loggersEndpoint;
    
    @PostMapping("/debug")
    public String enableDebug(@RequestParam String packageName, 
                              @RequestParam int minutes) {
        // 临时开启 DEBUG，指定时间后自动恢复
        loggersEndpoint.setLogLevel(packageName, LogLevel.DEBUG);
        
        ScheduledExecutorService executor = Executors.newSingleThreadScheduledExecutor();
        executor.schedule(() -> {
            loggersEndpoint.setLogLevel(packageName, LogLevel.INFO);
            System.out.println("日志级别已自动恢复为 INFO");
        }, minutes, TimeUnit.MINUTES);
        
        return packageName + " DEBUG 已开启，" + minutes + "分钟后自动恢复";
    }
}
```

## 🔥 扩展追问 4：安全控制与端点保护

```java
@Configuration
@EnableWebSecurity
public class ActuatorSecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .securityMatcher(EndpointRequest.toAnyEndpoint())
            .authorizeHttpRequests(auth -> auth
                .requestMatchers(EndpointRequest.to("health", "info")).permitAll()
                .requestMatchers(EndpointRequest.to("metrics", "prometheus")).hasRole("MONITOR")
                .anyRequest().hasRole("ADMIN")
            )
            .httpBasic(Customizer.withDefaults());
        return http.build();
    }
}
```

```yaml
# 敏感端点配置
management:
  endpoints:
    web:
      exposure:
        include: "*"  # 暴露所有端点
      base-path: /admin  # 修改基础路径
  endpoint:
    health:
      show-details: when_authorized  # 认证后才显示详情
    heapdump:
      enabled: false  # 禁用堆 dump（敏感操作）
    shutdown:
      enabled: true  # 开启优雅停机（需配合 security）
```
